# PetHouse com Tailwind CSS

Projeto desenvolvido durante a sexta semana da capacitação da área de Projetos Desktop da Asimov Jr.

O objetivo desta atividade foi aprofundar os conceitos fundamentais do **Tailwind CSS** por meio da adaptação de um layout existente para um novo contexto. Durante o desenvolvimento, o projeto original de catálogo de imóveis foi transformado na plataforma **PetHouse**, voltada para adoção de animais, aplicando conceitos de estilização com classes utilitárias, responsividade, organização de layouts com Flexbox e Grid, além da criação de efeitos visuais e transições utilizando exclusivamente recursos do Tailwind CSS.

---

## Tecnologias utilizadas

- HTML5
- Tailwind CSS

---

## Funcionalidades adicionadas

-  Adaptação completa da temática para uma plataforma de adoção de pets.
-  Sidebar personalizada com categorias de adoção.
-  Barra de pesquisa para encontrar animais.
-  Cards personalizados contendo:
  - Foto do pet;
  - Nome;
  - Sexo;
  - Idade;
  - Raça;
  - Porte;
  - Características;
  - Abrigo responsável.
-  Substituição das informações de imóveis por dados de ONGs e abrigos.
-  Efeitos de hover, zoom nas imagens e transições utilizando Tailwind CSS.
-  Layout totalmente responsivo para diferentes dispositivos.

---

## Conceitos praticados

- Flexbox
- CSS Grid
- Responsividade
- Hover e transições
- Componentização de layouts
- Utilização de utilitários do Tailwind CSS

---

##  Resultado

O projeto demonstra como um layout pode ser reutilizado e adaptado para um novo contexto, aplicando os conceitos de Tailwind CSS para criar uma interface moderna, responsiva e voltada para adoção responsável de animais.
